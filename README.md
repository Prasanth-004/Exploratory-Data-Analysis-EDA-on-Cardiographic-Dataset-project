# Exploratory Data Analysis (EDA) on Cardiographic Dataset

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a Cardiographic dataset to understand the characteristics, quality, and relationships within the data. The project includes data cleaning, statistical analysis, visualization, outlier detection, and correlation analysis to extract meaningful insights and prepare the dataset for machine learning applications.

---

## 🎯 Objectives

* Load and explore the cardiographic dataset.
* Clean and preprocess the dataset.
* Generate descriptive statistical summaries.
* Visualize numerical and categorical variables.
* Detect outliers and analyze feature distributions.
* Study relationships between variables using correlation analysis.
* Prepare the dataset for predictive modeling.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Description

The dataset contains cardiographic measurements collected from patients. It includes multiple numerical and categorical attributes that can be analyzed to understand patterns related to cardiovascular health and support future predictive analytics.

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported the cardiographic dataset.
* Explored dataset structure and dimensions.

### 2. Data Cleaning and Preparation

* Checked for missing values.
* Handled missing data appropriately.
* Identified and removed duplicate records.
* Verified data types and consistency.

### 3. Statistical Summary

Generated descriptive statistics including:

* Mean
* Median
* Standard Deviation
* Minimum and Maximum Values
* Quartiles
* Interquartile Range (IQR)

### 4. Data Visualization

Created visualizations to understand the dataset:

* Histograms
* Boxplots
* Bar Charts
* Pie Charts

These visualizations helped identify:

* Data distribution
* Skewness
* Outliers
* Category frequencies

### 5. Correlation Analysis

* Computed the correlation matrix.
* Visualized relationships using a heatmap.
* Identified strong positive and negative correlations between variables.

### 6. Outlier Detection

* Used boxplots and IQR techniques to identify extreme values.
* Evaluated the impact of outliers on the dataset.
